2. Description of Service

Smart Task Manager is a free browser extension that provides:

  • Task creation, management, and organisation with priorities, due dates, and recurrence
  • Calendar view, focus mode (Pomodoro timer), and productivity insights
  • Customisable themes and display options
  • Optional cloud synchronisation via Google account (Firebase/Firestore)
  • Desktop notifications and scheduled reminders
  • Image attachment via third-party image search (requires your own API key)
  • Data export and import as JSON

The Extension is provided at no charge. We reserve the right to introduce optional paid features in the future; free functionality will remain available.
